# Qu'est ce qu'un appel décliné ?

Les Expert-Stats opèrent une distinction entre appel **abandonné** et appel **décliné** : l'abandon est à l'initiative de l'appelant, alors qu'un appel décliné est rejeté par une ressource du PBX.

Par exemple, un agent qui laisse sonner jusqu'à expiration du timeout donnera lieu à un appel décliné ; il en est de même si l'agent refuse explicitement l'appel via le softphone.

Une file d'attente déclinera l'appel, vers une file de débordement ou la destination de dissuasion configurée, si aucun agent n'est disponible.
