# Pourquoi la durée d'attente de mes files est particulièrement longue ?

Il est possible que votre file d'attente joue un message de pré-décroché. Par défaut, le temps de ce pré-décroché est comptabilisé dans la durée d'attente de la file.

En passant par le menu "Configuration", puis "Files d'attente", il vous sera possible de définir la durée de ce message de pré-décroché pour une, plusieurs ou l'ensemble de vos files. 

Cette configuration aura pour effet :

1. de diminuer la durée d'attente de la file de la durée du message de pré-décroché.
2. d'indiquer séparément le nombre d'appels abandonnés total d'une part, le nombre d'appels abandonnés pendant le pré-décroché de l'autre.
